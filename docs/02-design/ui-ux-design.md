## 7. Accessibility (A11y)

### 7.1 Requirements

| Requirement | Implementation |
|-------------|----------------|
| **Keyboard navigation** | Tab order, focus indicators |
| **Screen readers** | ARIA labels, semantic HTML |
| **Color contrast** | WCAG AA (4.5:1 for text) |
| **Font sizing** | Respects user preferences, min 16px |
| **Touch targets** | Min 44x44px for mobile |
| **Reduced motion** | Respects prefers-reduced-motion |

### 7.2 Elderly-Friendly Features

| Feature | Implementation |
|---------|----------------|
| **Large text option** | Toggle in settings |
| **High contrast mode** | Dark backgrounds, light text |
| **Simple navigation** | Clear labels, minimal depth |
| **Large buttons** | h-12 (48px) option |
| **Clear error messages** | Vietnamese, helpful |
